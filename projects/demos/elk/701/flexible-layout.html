💡 通用计算方法说明：
所有层的计算都基于 originX 作为参考点：

• 靠左对齐：x = originX + padding + nodeIndex * (width + spacing)
• 居中对齐：x = originX + (referenceWidth - nodeWidth) / 2
• 靠右对齐：x = originX + referenceWidth - padding - totalWidth + nodeIndex * (width + spacing)

其中 referenceWidth 是第1层节点的宽度，作为后续层的参考基准。
